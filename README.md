# What is it?  
Discord detects you playing a game as long as the exe has the correct name and it is in the correct path.  
With this knowledge, you can complete Discord's 'play' quests without installing the game/app.  
**This app helps you automate the task of creating a dummy exe in the correct path of a game and run it** using your input.  
  
For example, Wuthering waves' exe path is:  
any folder\ `Wuthering Waves Game\Client\Binaries\Win64\Client-Win64-Shipping.exe`
You just have to input this path and the app will create these inside of the app's directory\DQC Game Folders\..

Guide on how to use it is provided inside of the app
